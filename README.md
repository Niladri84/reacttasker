# Todo App

A clean, modern task management application built with React and Vite.

## Framework

Vite + React 19

## Features

- Add, complete, and delete tasks
- Task statistics (total, active, completed)
- Clear completed tasks
- Local storage persistence
- Responsive design
- Clean Light theme with minimal aesthetic

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

Clean Light - A minimal, modern aesthetic with light gray backgrounds (#F8F8F8), white cards with soft shadows, blue accent colors for primary actions, and subtle hover states for a calm, professional experience.
